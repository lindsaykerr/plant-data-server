Welcome to Plant data server API decription

Here you can find the API documentation for the plant data server

The following list the API paths with descriptions:

  • /api/v1/submit Submits sensor data to the server (POST METHOD)
  • /api/v1/data/{plant id}/latest Get the latest data for a plant (GET METHOD)
  • /api/v1/reset/db Wipes testing database of sensor data (DELETE METHOD)
